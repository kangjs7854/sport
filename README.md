## vue + express + mongodb + docker 部署的前后端分离的项目
[在线预览](http://175.24.20.162:8080 "hupu") 
> 实现的功能
1. pc和移动端的兼容
2. 新闻和体育资讯内容的爬取和展示
3. 运动装备的购买（支付流程未完成）

> 目录结构

  - server express
  - docker-compose.yml  docker的快速部署文件
